Bem vindo ao road map do JAVA

Java é uma linguagem de programação orientada a objetos, portável e multiplataforma, criada pela Sun Microsystems em 1995 (atualmente mantida pela Oracle). É amplamente utilizada no desenvolvimento de aplicações corporativas, aplicativos Android, sistemas embarcados, jogos e softwares de grande escala. Um dos principais lemas do Java é: "Escreva uma vez, execute em qualquer lugar" (Write Once, Run Anywhere – WORA), graças à Java Virtual Machine (JVM), que permite executar o mesmo código em diferentes sistemas operacionais.
